# output/

Esta carpeta contiene los artefactos generados por `proyecto_big.py`.

## Salidas principales
- `market_merged.csv`: dataset consolidado ventas + especificaciones
- `kpis_by_brand.csv`: KPI agregado por marca
- `kpis_by_model.csv`: KPI agregado por modelo
- `kpis_by_city.csv`: KPI agregado por ciudad (si aplica)
- `kpis_by_year.csv`: KPI agregado por año (si aplica)
- `correlation_matrix.csv`: matriz de correlación de variables numéricas

## Uso sugerido
Consumir estos archivos para crear dashboards, reportes ejecutivos y análisis comparativos adicionales.
